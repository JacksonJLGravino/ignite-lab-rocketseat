# Ignite Lab Rocketseat

Este repositório reúne os projetos que participei durante o Ignite Lab da Rocketseat.

A ideia foi centralizar tudo em um único lugar para facilitar a visualização e mostrar a evolução ao longo do evento.

## O que está aqui

- Design System: um projeto com componentes reutilizáveis e interface moderna.
- Microservices: uma implementação backend com foco em arquitetura e organização.
- Plataforma de Evento: uma aplicação para exibir e gerenciar conteúdos de evento.
- RocketHelp: um projeto mobile voltado para ajuda e gestão de demandas.

## Tecnologias envolvidas

- React
- TypeScript
- Node.js
- NestJS
- Prisma
- Tailwind CSS
- GraphQL
- React Native

## Como navegar

Cada projeto está em uma pasta separada dentro deste repositório. Basta entrar na pasta desejada para acessar o conteúdo específico de cada um.

Obrigado por visitar!
